자바스크립트에서의 데이터 입/출력

1. 데이터를 출력하는 구문

1) [window.]alert("알람창에 출력할 문구");
2) window.console.log("콘솔창에 출력하고 싶은 문구");
3) document.write("화면상에 출력할 문구");
4) 선택한요소.innerHTML | innerText = "요소에 출력할 문구";

3) document.write("화면상 출력할 문구")



4) 선택한요소.innerHTML | innerText = "요소에 출력할 문구";

자바스크립트에서 어떤 요소(element)안의 값(content)을 가지고 온다거나
변경하고자 할 떄 innerHTML, innerText라는 속성에 접근

2. 데이터 입력 받는 구문(변수에 기록 가능)

1) 변수 = [window.]confirm("질문내용");
2) 변수 = [window.]prompt("질문내용");
3) 변수 = 선택한요소.속성(id, className, innerHTML, innerText,..)
4) 변수 = 선택한input요소.value

1) [window.]confirm("질문내용");

confirm 호출시 "질문내용"과 "확인/취소" 버튼이 존재하는 알림창 발생
확인버튼 클릭시 true, 취소버튼 클릭시 false를 반환


[window.]prompt("질문내용");

prompt 호출시 "질문내용"과 입력할 수 있는 "텍스트 상자"와 "확인/취소" 버튼이 보여지는 알람창 발생
확인 버튼 클릭 시 텍스트 상자에 입력되어 있는 값이 문자열로 반환(아무리 숫자가 써있어도 문자열), 취소버튼 클릭시 null을 반환(글자입력을 했어도 취소를 누르면 무조건 null)



4) 선택한 input요소.value

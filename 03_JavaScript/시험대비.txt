1. css 파일 => 외부스타일 연결방법(html과 외부 css파일간의)
Inline Style Sheet=> HTML 태그의 style 속성에 CSS 코드를 넣어 적용시키는 방법
Internal Style Sheet=> HTML 문서 안에 스타일 코드를 넣는 방법(<style>과 </style> 안에)
Linking Style Sheet=> 별도의 CSS 파일을 만들고 HTML 문서와 연결하는 방법
<link rel="stylesheet" href="style.css"> (style은 css파일 파일명, css는 확장자명)

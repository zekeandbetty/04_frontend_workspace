CSS 기본선택자

선택자란?

특정 html요소를 선택하고자 할 떄 사용하는 기능
해당 요소를 선택해서 원하는 "스타일"과 "기능"을 적용시킬 수 있음

1. 모든(전체) 선택자 : *

현재 이 문서상에 모든 요소들을 다 선택하고자 할 떄 사용

    * {
        스타일속성: 값; 
        스타일속성: 값;

    }

2. 태그 선택자 : 태그명

현재 이 문서상에 존재하는 해당 태그들을 다 선택하고자 할 떄 사용

        태그명 {
            스타일속성 : 값;
        }

3. 아이디 선택자 : #아이디명

현재 이 문서상에 특정 html요소 "하나"만을 선택하고자 할 떄 사용 단, 해당 요소에 id속성을 이용해서 고유한 아이디값 부여해야함 

        #아이디명{
            스타일속성 : 값;
        }

4. 클래스 선택자 : .클래스명

해당 이 문서내에 내가 원하는 요소 "여러개"를 선택하고자 할 떄 사용 

        .클래스명 {
            스타일속성 : 값;
        }

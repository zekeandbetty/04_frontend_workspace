jQuery 개요

* jQuery란?

기존에 자바스크립트 언어만으로 구현하기엔 복잡했던 구문 작성들을
간소화 하기 위해 "존 레식"에 의해 만들어짐
javaScript 기반 라이브러리(유용한 기능들의 모음집) => ex_ojdbc6.jar 같은거

복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
(적은 양의 코드로 빠르고 풍부한 기능 제공) >> jQuery 장점
1) DOM요소 => html 태그들.. 와 관련된 스크립트를 보다 쉽게 구현할 수 있음

2) Ajax(비동기식 통신), 이벤트 처리 등등 폭넓게 지원함

3)jQuery와 관련된 확장형 플러그인, 오픈 api 등을 지원

* jQuery 라이브러리 연결 방법

head태그 내에 script 태그로 외부 jQuery 관련 .js 파일 연결하기
단, 오프라인 / 온라인 방식 있음

- jQuery 공식 사이트 : 공식사이트

1. jQuery 라이브러리 직접 다운로드 받은 후 경로 지정(오프라인) 2. CDN(Content Delivery Network) 방식(온라인)
파일을 다운로드 받아서 저장할 필요 없이 라이브러리를 제공하는 사이트의 url 제시해서 파일 경로를 지정

* jQuery 구문 작성하기

jquery 구문으로 DOM 요소를 다루는 구문은
문서상에 요소들이 다 만들어지고 나서 실행되어야함

> 순수 자바스크립트에서는 

                window.onload = function(){
                     실행내용;
                }

                javaScript에서의 window.onlaod 구문은
                해당 문서 요소들이 다 로드되고(그려지고, 렌더링되고) + 
                외부문서들(css 파일)도 다 가지고 온 뒤에 실행됨
                그리고 window.onload 구문은 해당 문서에서 단 한번만 작성 가능
                (여러개 있다면 마지막에 작성된것만 반영, 나머지는 무시됨)
> jQuery에서의 해당 내용은 3가지 구문으로 제공 
                1) jQuery(document).ready(function(){
                    실행내용;
                });

                2) $(document).ready(function(){
                    실행내용;
                });

                3) $(function(){
                    실행내용;
                });

                해당 문서의 요소들만 다 로드되면(그려지면, 렌더링되면) 곧바로 실행됨
                여러번 기술 가능함(제한없음)=> 순서대로 실행
